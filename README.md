# Reg4IA 2026 - Materiale 
Benvenuti nel repository ufficiale della tre giorni di lavoro nell'ambito dell'iniziativa **Reg4IA**, che si tiene a Bari (7-9 luglio 2026) dedicata all'adozione consapevole, etica ed inclusiva dell'Intelligenza Artificiale nella Pubblica Amministrazione.

## 📦 Come scaricare i materiali
Per scaricare tutti i documenti in un unico file compresso (.zip):
1. Cliccate sul pulsante verde in alto a destra **"Code"**.
2. Selezionate **"Download ZIP"**.

In alternativa, potete navigare nelle cartelle qui sopra e scaricare i singoli file PDF o slide di vostro interesse.
